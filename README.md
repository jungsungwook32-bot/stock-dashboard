# 개인 주식 관리

## 폴더 구조

```
주식/
├── portfolio/          # 포트폴리오 관리
│   ├── holdings.md     # 현재 보유 종목
│   ├── transactions.md # 매매 기록
│   └── watchlist.md    # 관심 종목
├── markets/            # 시장별 종목 정보
│   ├── korea/          # 한국 시장
│   │   ├── kospi.md    # KOSPI 주요 종목
│   │   ├── kosdaq.md   # KOSDAQ 주요 종목
│   │   └── etf.md      # 한국 ETF
│   ├── us/             # 미국 시장
│   │   ├── nasdaq.md   # NASDAQ 주요 종목
│   │   ├── nyse.md     # NYSE 주요 종목
│   │   ├── sp500.md    # S&P 500 정보
│   │   └── etf.md      # 미국 ETF
│   └── global/         # 글로벌 (세계 500위 기업)
│       └── top500.md   # 한국/미국 제외 글로벌 기업
├── analysis/           # 분석
│   ├── sector.md       # 섹터별 분석
│   ├── dividend.md     # 배당 분석
│   └── notes.md        # 투자 메모
└── reference/          # 참고 자료
    ├── terms.md        # 용어 정리
    └── calendar.md     # 주요 일정
```
